## Features
### Dự án có những tính năng chính như sau:
- CRUD: thêm sửa xóa sản phẩm
- Đăng ký, đăng nhập (hash password, JWT)
- Tìm kiếm sản phẩm
- Thêm sửa xóa địa chỉ giao hàng
- Bình luận, đánh giá sản phẩm
- Đặt hàng
